# Screenshots Placeholder

This directory will contain screenshots of the application.

## Required Screenshots:

1. **home.png** - Main landing page with feature cards
2. **login.png** - Login/authentication page

## How to Add Screenshots:

1. Take screenshots of each page at a resolution of at least 1920x1080
2. Save them with the filenames listed above
3. Place them in this directory
4. The README will automatically display them

## Tips for Great Screenshots:

- Use demo data that looks realistic
- Ensure the UI is in a clean state (no console errors)
- Capture the full page or the most important sections
- Use consistent browser width for all screenshots
- Consider using both light and dark mode screenshots
